# Research-Paper-Analysis-System

An intelligent research paper analysis system for automated domain classification, keyword extraction, TLDR generation, and semantic paper recommendation using modern NLP techniques.

## Tech Stack

- ModernBERT (Fine-Tuned)
- KeyBERT
- DistilBART
- BAAI/bge-small-en-v1.5
- FAISS
- FastAPI
- PyTorch
